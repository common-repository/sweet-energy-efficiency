Graphically Visually present Energy Efficiency 
Class / Label / Rating / Scale with related consumption values

== Description ==

This WordPress Plugin will Graphically Visually present Energy Efficiency 
Class / Label / Rating / Scale with related consumption values, ideally to use on real estate 
websites/portals but also for any other purpose where Energy Efficient is very important like Cars, 
Ait Conditioning devices, Fridges etc.

Best to use with themes and plugins made by our team: <a href="https://wpdirectorykit.com/">wpdirectorykit.com</a>

Extra customizable, select any colors, units, values so suitable for any standard or even for different purpose.

Features:

*  Shortcodes
*  Widgets
*  Elementor elements
*  Customizable colors, values, numbers
*  Support multiple templates/graphics designs

== Installation ==

1. Upload `sweet-energy-efficiency.zip` to the `/wp-content/plugins/` directory and extract
2. Activate the plugin through the 'Plugins' menu in WordPress
